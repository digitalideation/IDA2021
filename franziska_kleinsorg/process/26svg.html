IDA Digital Design, Day 14

SVG

Today we will work in small breakout rooms on our own SVG assignments while I meet individually with each student.

  • Feel free to read or watch the below resources together and help each other out.

  • Please also take this opportunity to share your main projects with your small group and get feedback.

Today's Assignment:

  • View the resource material below as needed and then...
  • Create a web page (or series of pages) made up entirely of SVG. (Yes, additional markup can be used if needed—as containers, or to create links between pages for example.)
  • It can be separate from—but should be related to—your main project. For example, it should be inspired by your song, mood board, project idea, etc.
  • Use the web technology skills you've learned so far to add interactivity to the page.
  • If you're comfortable enough, try out more advanced techniques such as filters or animation. But don't worry! We will get to animation next week. ðŸ˜Š
  • Important: Upload your assignment to GitHub by 16:00 today so Hanna and I can review it.
