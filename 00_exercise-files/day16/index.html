Scrollytelling

Scrollytelling is visual storytelling for the web: a powerful technique based on a simple concept: new content and visuals appear or change through transitions as users scroll down or up the web page.

You will find lots of examples! Two sophisticated ones:

https://panamaalbrown.com/
https://www.sbs.com.au/theboat/

Most sites make use of these technique, parallax to create an effect of depth and viewport detection to trigger animation on an element when the user arrives there.

(Pure CSS) Parallax Effect

Parallax scrolling is one of those web design trends that just keeps on going, and there are sites that use it to brilliant effect. The technique involves designing the background of a website layout to move at a slower rate than the foreground when the user scrolls, creating a 3D-like effect.

W3C very simple background-attachment: fixed;
https://www.w3schools.com/howto/howto_css_parallax.asp

Demo Using CSS and perspective:
https://keithclark.co.uk/articles/pure-css-parallax-websites/demo3/

Tutorial using CSS translateZ and perspective:
