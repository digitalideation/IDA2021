GPU Acceleration

Basically: GPU's are really powerful these days. If you can offload the processing of motion on your web page to the GPU, it will perform more smoothly. You can do that by "promoting" an element to a compositing layer by using CSS 3D transforms. What we're trying to do is avoid "repaints" (when the browser has to repaint the whole page because something changed).
